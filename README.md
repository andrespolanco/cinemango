# Ejecutar desarrollo
1. Clonar repositorio
2. Ejecutar

```
npm install
```

3. Tener nest cli instalado
```
npm i -g @nestjs/cli
```
4. Levantar la bd
```
docker compose up -d
```
## Stack usado
* MongoDB
* NestJS
